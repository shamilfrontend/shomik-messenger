# ESLint Configuration

Проект использует ESLint с правилами от Airbnb для Frontend и Backend.

## Использование

### Frontend
```bash
cd frontend
yarn lint          # Проверить и автоматически исправить ошибки
yarn lint:check    # Только проверить без исправления
```

### Backend
```bash
cd backend
yarn lint          # Проверить и автоматически исправить ошибки
yarn lint:check    # Только проверить без исправления
```

## Конфигурация

- **Frontend**: `.eslintrc.cjs` в папке `frontend/`
- **Backend**: `.eslintrc.cjs` в папке `backend/`

Оба проекта используют:
- `eslint-config-airbnb-base` - базовые правила Airbnb
- `@typescript-eslint` - правила для TypeScript
- `eslint-plugin-import` - правила для импортов
- `eslint-plugin-vue` - правила для Vue (только Frontend)

## Исключения

Файлы в `.eslintignore` не проверяются линтером.
